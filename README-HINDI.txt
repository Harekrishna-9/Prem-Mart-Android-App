App Name: Prem Mart
Package Name: com.premmart.store

APP AUTO UPDATE KAISE HOGA:
Ye app aapki live GitHub Pages website ko load karta hai. Aap GitHub par website ka HTML, CSS, JavaScript, product, price ya Firebase data update karenge to app kholne/refresh karne par wahi updated website dikhegi. Har website change ke liye naya APK banana zaroori nahi hai.

APK BANANE KA SABSE AASAAN TARIKA — GITHUB:
1. GitHub par ek naya repository banayein, jaise Prem-Mart-Android-App.
2. Is ZIP ke andar wale Prem-Mart-Trial-App folder ki saari files repository ke root me upload karein.
3. Repository me Actions tab kholein.
4. “Build Prem Mart APK” workflow select karein.
5. Run workflow dabayein.
6. Build complete hone ke baad neeche Artifacts me “Prem-Mart-Trial-APK” download karein.
7. ZIP extract karein; uske andar app-debug.apk milega.
8. APK mobile me bhejkar install karein. Zarurat pade to Install unknown apps permission allow karein.

ANDROID STUDIO SE:
1. Android Studio install karein.
2. Prem-Mart-Trial-App folder ko Open Project se open karein.
3. Gradle sync complete hone dein.
4. Build > Build Bundle(s) / APK(s) > Build APK(s) select karein.
5. APK app/build/outputs/apk/debug/app-debug.apk me milega.

INCLUDED FEATURES:
- Live Prem Mart website WebView me open
- Website updates app me automatically visible
- JavaScript, Firebase aur local storage support
- File/image upload support
- Current location permission
- Android back-button navigation
- Loading progress bar
- WhatsApp, phone aur map links support

IMPORTANT:
Ye trial app project hai. Debug APK testing ke liye hai. Play Store ke liye baad me signed release AAB, privacy policy, app icon/screenshots aur Play Console setup karna hoga.
